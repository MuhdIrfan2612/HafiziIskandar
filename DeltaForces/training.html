   TRAINING & DEVELOPMENT  
   
 
 

All Security Officers will receive comprehensive training from DFSS Training Manager. DFSS Training Manager will meet with Multimedia Development Corporation Security Manager and present training material for review. All Security Officers will undergo, “On Job Training” (OJT) to ensure they fully understand their duties, responsibilities and SOP’s.

Module 1
 
 
Company background  
Security attire and appearance  
Communication skills  
Preparation of Report (logbook, station diaries…etc)  
ID Badge policy and procedures  
Access Control  
Under carriage inspection  
Roll call / briefing session  
Security officers requirements  
Damage Report and Inspection  
Loss Inspection  
Challenge Procedures  
Fire Fighting usage  
     

Module 2

Security Incident report  
Incoming parcels/packages/documents/bulk cargoes  
Patrolling / Guard Tour  
Vehicle Inspection  
Key Management system  
Usage of equipment (W/Talkie, touch probe, baton…etc)  
Lost and found items  
Security seal control  

Module 3

Emergency responses (fire, bomb threat, work place violence, robbery)  
Event Management and Crowd Control  
Leadership roles and responsibilities  
C-TPAT  
TAPA’s requirement  
ISO 28000: Security
